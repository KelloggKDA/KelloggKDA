R

Getting Started

R is a free, open source tool available for data scientists, business analysts, and everyone in between. You’ll need two things to get started: * Download R: this is the core package that will let you run and test R packages * Download R Studio: this is a great user interface to R that makes it easier to quickly test and deploy R scripts

General Resources

These are some recommended resources in order to quickly self-learn and get up to speed:

  • CodeSchool R Tutorial : An online interactive tutorial that requires no downloads and will walk you through the basics of data analysis in R. Great for beginners and people completely new to the tool.
  • R for Data Science: A complete online book about using R for data science. This covers all the basics of data science, as well as providing complete examples and demos
  • swirl: swirl is actually a package in R that teaches you how to program and fundamentals of data science within your console. If you want to learn R, or even brush up on your skills, try it out.
